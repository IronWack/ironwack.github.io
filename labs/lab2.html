Mobile Application Tool

Information Description
Tool Name Andriod Studio
Platform Andriod
Application Development Type Native, this is good and bad. Good being that the apps made for the specific platform should work more seemlessly but bad in the sense that you can only use the app on one platform unless the developers make a port.
Website Andriod Studio
Release Date December 2014.
Developers Google and JetBrains.
License Freeware + Source code.
Price FREE!!!(everyone loves FREE).
Programming Language(s) Java and C++, but they support many extensions.
Features I will list some of the newest featers: Code layout manager, database inspector, and offline mode.
Debugger The debugger had breakpoints and a debugger log, as well as extensions that handle debugging formatting and suggestions.
Built-in Emulator The built-in emulator allows for quick swapping from different versions of andriod models as well as resolution scale and oritentation. You can do pretty much do anything a normal phone can.
Host machine requirements The basic requirements:
Microsoft Windows 7/8/10.
4 GB RAM.
2 GB of disck space, 4 GB for SDK and emulator.
1280 x 800 minimum screen resolution.
SDK Tools 26.1.1.
64-bit processor.
CPU with UG support.
HAXM 6.2.1 or later.
IDE Does not need any specific IDE can you notepad++ or sublime.
Software to be installed Nothing other then the host machine requirements.
Advantages Faster Deployment
Accurate programming
Faster testing and programming
Inclusive App development
Better App indexing
Disadvantages High amounts of RAM needed to run smooth
Updates to Gradle files can be hectic at times
Multitasking is hard to do with Andriod studio
Avaliable languages Andriod extension that allows for quick and accurate languages translation from almost all widely spoken languages.
Uploading and selling apps At a minimum you need to remove Log calls and remove all andriod:debuggable calls after that its as simple as pick where you wish to release you app. Andriod studio takes no money.
Third Party Emulator From what I can see there is only one third party emulator avaliable for andriod studio and it does not seem worth it. Its Genymotion, you need to install Oracle VM VirtualBox and then install Genymotion andriod device emulator. Next you need to download the andriod studio plugin that lets Genymotion integrate into it, after that you are done. I will say though that the added RAM and CPU usage from the VM is more then the already integrated emulator, not exaclty sure why you would want this.

Number of apps avaliable 2.96 Million, wow OwO
